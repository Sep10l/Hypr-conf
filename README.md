  # Description

my Config/s for arch linux Hyprland, binds and interface
  Hope you enjoy, feel free to redistribute.

  # Additional

make sure to source the file in your hyprland.conf file if you do not want to import it directly

  # Installation

-install the raw files, somewhere you can remember.
  in the hyprland.conf file source your downloaded .conf files. For example:

  source = ~/.config/hypr/scripts/startup.conf, you have to source them in the appropiate location.
    Alternatively you can download the hyprland.conf file with and overwrite it with the source
